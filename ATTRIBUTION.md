### What We Use

NthLayer includes 46 alert template files sourced from awesome-prometheus-alerts, covering:
- Databases (PostgreSQL, MySQL, Redis, MongoDB, Elasticsearch, etc.)
- Message Brokers (Kafka, RabbitMQ, NATS, Pulsar)
- Load Balancers (Nginx, HAProxy, Traefik, Apache)
- Orchestrators (Kubernetes, Nomad, Istio, Linkerd)
- Observability (Prometheus, Loki, Thanos, Cortex)
- And many more (46 technologies total)

### How We Use It

1. **Templates are loaded** from `src/nthlayer/alerts/templates/`
2. **Customized per service** with:
   - Service name labels
   - Team labels
   - Tier labels (critical/standard/low)
   - Runbook URLs
   - Notification channel annotations
3. **Original alert logic preserved** - expressions and thresholds unchanged
4. **Generated with attribution** - all output files include CC BY 4.0 attribution

### Changes Made

- **No changes to alert logic** - Original PromQL expressions preserved
- **Additions only:**
  - Service context labels
  - Team metadata
  - Runbook URL annotations
  - Tier-based filtering
